# CRUD com Laravel 5.4 e Angular 1.6.3

# Instalação

* git clone https://github.com/HermesMartins/crud-laravel-5.4-angularjs-1.6.3.git nomedoprojeto
* cd nomedoprojeto
* composer install
* crie uma database e configure o arquivo .env
* php artisan key:generate
* php artisan migrate
* php artisan db:seed
* php artisan serve
