# CRUD with Laravel 5.3 and VueJS 2.0

# Installation

* git clone https://github.com/HermesMartins/crud-laravel-5.4-angularjs-1.6.3.git projectname
* cd projectname
* composer install
* create a database and setup .env
* php artisan key:generate
* php artisan migrate
* php artisan serve
